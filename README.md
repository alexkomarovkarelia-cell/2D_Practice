# 2D Practice

Учебный 2D-проект на Unity, который я делаю в рамках курса.  
Цель проекта — пройти путь от базовой настройки сцены и управления до врагов, анимаций, боевой логики, сборки и подготовки к релизу.

Проект используется как практика:
- Unity 2D
- C#
- New Input System
- работа с анимациями
- работа с префабами
- организация структуры проекта
- базовая логика врагов
- подготовка фундамента для более серьёзных проектов

---

## Что уже реализовано

### Игрок
- движение игрока
- поддержка нескольких режимов управления:
  - Keyboard
  - Joystick
  - Auto
- спринт
- работа через **New Input System**
- поддержка экранного джойстика
- базовая анимация движения

### Карта и сцена
- настроена 2D-сцена
- используются слои карты, стен и декора
- организована базовая структура проекта

### Враги
- реализовано простое преследование игрока
- враги двигаются к игроку
- добавлены анимации врагов
- используется 4 направления движения:
  - вверх
  - вниз
  - влево
  - вправо
- сделано несколько типов врагов на одной системе логики
- враги отключаются, если игрок уходит слишком далеко

### Архитектура / организация
- проект разложен по папкам
- сторонние ассеты вынесены отдельно
- используется **Zenject / Extenject**
- ссылка на игрока передаётся через installer
